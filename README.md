## 使用方法

### 下载链接

- [Github下载](https://github.com/zwh8800/RGTV/releases/download/v1.0.0/RGTV.zip)
- [百度网盘下载](https://github.com/zwh8800/RGTV/releases/download/v1.0.0/RGTV.tar.gz)

### 安装
- 解压RGTV.zip
- 将解压后所有文件复制到 内存卡\\Roms\\APPS 下
- 在RG40XX上打开RGTV
